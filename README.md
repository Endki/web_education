# 마이 프로젝트
`개인 프로젝트` `웹기술` `내가 하고 싶은 것들`

내가 개발해보고 싶은 기능들을 모아서 커뮤니티 형식으로 구현

### 구현 기능

- JavaScriptGame (지뢰찾기, 틱택토 등등)
- 게시판 (커뮤니티 기능)
- 회원가입 / 로그인
- 스케쥴러
